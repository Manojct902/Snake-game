🐍 Snake Game (JavaScript)

A simple and interactive Snake Game built using HTML, CSS, and Vanilla JavaScript.

This project demonstrates DOM manipulation, game logic implementation, grid systems, event handling, and local storage usage in JavaScript.

🎮 Features

🐍 Smooth snake movement

🍎 Random food generation

📈 Score tracking

🏆 High score saved using LocalStorage

⏱️ Live timer

🔁 Restart game functionality

🎨 Responsive grid layout using CSS Grid

🪟 Start and Game Over modal screens

🛠️ Technologies Used

HTML5

CSS3 (Flexbox + Grid)

JavaScript (ES6)

LocalStorage API
